# HarmonizeMe
Web app to make chordal harmony to accompany a user sung input

Libraries:
flask 0.12.1;
librosa 0.4.2;
aubio 0.4.5;
numpy 1.12.1;
matplotlib 2.0.0;

Running the flask app on local:
After cloning:

cd HarmonzieMeApp/flaskapp/
export FLASK_APP="flask_run.py"
flask run

Then go to 127.0.0.1:5000/ on Chrome.
